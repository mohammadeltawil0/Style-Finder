# Welcome to Style Finder

## Get started

1. Install dependencies

   ```bash
   cd frontend
   npm install
   ```

2. Start the app

   ```bash
   npm run start
   ```

3. Download Expo Go on your phone, and use a camera to scan the QR code that shows in the terminal. 

4. This will show you our app. Once you load in, via terminal, choose log in anonymously if given the option.
